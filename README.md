<h1 align="center">
     WeatherApp
</h1>



This project allows you to check the weather and keep notes based on the weather.

## Using Technologies:
- React Native
- React Native Navigation
- React Native AsyncStore
- React Native HeroIcons
- React Native Progress
- Nativewind
- Axios
- Lodash
- Zustand
